# 天猫用户复购行为数据分析项目
## 项目简介
本项目以 **SQL为核心**，基于阿里天池天猫用户行为公开数据集，
完成数据清洗、多表关联、用户分层统计、用户画像分析，
最终使用 Tableau 制作交互式可视化仪表盘，支持多维度筛选联动。

全程无Python，纯SQL+Tableau完成完整数据分析闭环。

## 工具技术
- 数据处理：MySQL、DBeaver
- 可视化工具：Tableau
- 数据来源：阿里天池公开数据集

## 项目内容
1. 数据清洗：SQL剔除空值、异常值、清理NULL无效数据
2. 多表关联：关联用户行为表、用户信息表、复购标签表
3. 维度分析：
   - 复购/非复购用户行为指标对比
   - 用户年龄段分布画像分析
   - 商家流量与转化统计
4. 可视化仪表盘：
   内置年龄、性别、复购标签筛选器，全图表联动交互

## 仓库文件说明
- SQL脚本(DBeaver)：项目所有核心查询SQL代码
- Tableau：打包工作簿.twbx + 仪表盘全景截图
- 项目分析报告：完整专业数据分析报告

## 项目亮点
1. 所有数据处理与统计均由SQL完成，逻辑可复现
2. Tableau仪表盘支持全局筛选、多图表联动
3. 分析结论贴合电商业务，可直接用于运营决策参考
